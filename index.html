What is janglipse?

About

Janglipse offers integration for jangaroo within Eclipse. It is based upon Xtext, a framework for domain-specific languages.

License

Janglipse is Licensed under the Eclipse Public License version 1.0. For this, you can use janglipse in order to implement software for personal and commercial use.
